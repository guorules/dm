
1. Create a folder (with your name) in https://github.com/guorules/dm/tree/master/project

2. Data Source: http://api.bitcoincharts.com/v1/csv/ and the column format is: (timestamp, price, volume).

3. Write a program (in any language) to do the following:
(a) Read in all the data from the data source.
(b) The ultimate goal is, for every token/coin, predict whether it's going up or down in the next 24 hours.
(c) Think about which models/algorithms could be the best fit for this problem.
(d) Implement them and adjust the paramenters (configurations).
(e) Define metrics and evaluate them.

4. Check in your program (code) to your folder and write the final results in a file called output.txt.
