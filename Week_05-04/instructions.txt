2. Write a program (in any language) to read in input.txt, and use either the Apriori algorithm or FPGrowth algorithm to find all the association rules (in the form of X->Y) with minimum support = 0.5 and minimum confidence = 0.5.
3. Check in your program (code) to your folder and write the final result (all the association rules) in a file called output.txt.
